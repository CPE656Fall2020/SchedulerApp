Class GreedyOptimizedLowPowerScheduler

GreedyOptimizedLowPowerScheduler solves the scheduling problem using a greedy algorithm to always pick the lowest power device for each pass that can compute the AES encryption problem within the allowed time window. If a solution can be found, it will be globally optimized to use the minimal amount of power, however, it is NOT guaranteed to be optimized to be the fastest. Alternative profiles may be possible to allow for solving the problem with the the same or greater level of energy (but NOT less), but at a faster rate.
